4. KEY FINDINGS
--------------------------------------------------------------------------------
- Dataset contains 21 different classes
- Classes are organized into 4 main categories:
  * Healthy rice plants
  * Rice diseases
  * Rice pests
  * Nutrient deficiencies
- Images vary in dimensions and quality
- Class imbalance present - need addressing during training

6. RECOMMENDATIONS FOR MODELING
--------------------------------------------------------------------------------
- Resize all images to consistent dimensions (e.g., 224x224 or 299x299)
- Apply data augmentation for minority classes:
  * Random rotation, flip, brightness adjustment
  * Random crop, zoom, translation
- Use transfer learning (ResNet, EfficientNet, Vision Transformer)
- Implement stratified train-validation-test split
- Consider class weights or focal loss for imbalanced classes
- Use appropriate metrics: F1-score, precision, recall per class
- Consider hierarchical classification (category → specific disease)
- Use PIL-based data loaders to handle Unicode file paths
